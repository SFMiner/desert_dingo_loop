# Audio Directory

Place your sound effect and music files here.

## Expected Sound Effects

| File | Description | Style |
|------|-------------|-------|
| `pop.ogg` | Placing organism | Satisfying, soft pop |
| `crunch.ogg` | Eating sound | Cartoonish crunch |
| `whistle_down.ogg` | Organism starving | Low pitch whistle, gentle |
| `success.ogg` | Healthy ecosystem | Cheerful chime |
| `click.ogg` | UI button click | Soft click |
| `day_start.ogg` | New day begins | Morning/sunrise feel |
| `win.ogg` | Game victory | Celebratory |
| `hover.ogg` | Mouse hover | Very subtle |

## Music (Optional)
| File | Description |
|------|-------------|
| `desert_ambient.ogg` | Gentle background loop |

## Format Recommendations
- OGG Vorbis format (best for web)
- 44.1kHz sample rate
- Mono for SFX, stereo for music
- Keep file sizes small for web loading

## Audio Design Notes

This game is designed for K-5 students, including those with special needs.
- **Avoid**: Harsh buzzers, alarms, or sudden loud sounds
- **Prefer**: Gentle, calming audio feedback
- **Volume**: Keep everything at moderate levels
